# 同事使用方式

如果同事打不开你电脑上的局域网地址，可以改成“每个人本地运行一份看板”。

## 怎么发给同事

把整个 `AI比赛` 文件夹压缩成 zip，发给同事，或放到公司共享盘。

同事解压后，双击：

```text
start-dashboard.bat
```

然后在浏览器打开：

```text
http://127.0.0.1:8787
```

命令行窗口不要关。关掉窗口后，看板服务也会停止。

## 如果提示没有 Python

这台电脑需要先安装 Python 3。安装后重新双击 `start-dashboard.bat`。

## 这种方式的特点

- 不需要连你的电脑。
- 不需要改你电脑防火墙。
- 每个人都在自己电脑本地使用。
- 每个人搜索出来的数据存在自己的电脑里，不是多人共用同一份记录。

## 如果要所有人共用同一个网址

需要把看板部署到一台大家都能访问的公司内网机器、云主机或带登录保护的服务入口上。这种方式更适合长期团队使用。
